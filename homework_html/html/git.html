Git

  • git init — создаёт новый репозиторий в текущей папке.
  • git status — показывает состояние репозитория и изменённые файлы.
  • git add . — добавляет все изменённые файлы в индекс (подготовка к коммиту).
  • git commit -m "message" — фиксирует изменения в репозитории.
  • git log — показывает историю коммитов.
  • git clone URL — клонирует удалённый репозиторий на компьютер.
  • git push — отправляет коммиты на удалённый репозиторий.
  • git pull — получает обновления с удалённого репозитория.
